Proyecto Totito

Descripcion del Proyecto

Este proyecto consiste en el desarrollo de un juego de "Totito" (también conocido como "Tres en Línea" o "Tic-Tac-Toe") utilizando el lenguaje de programación Java. El objetivo principal era el de crear un juego interactivo donde dos jugadores puedieran competir, alternando turnos, con una interfaz gráfica amigable y sencilla.

Tecnologías y Herramientas Utilizadas

Lenguaje de Programación: Java El juego está completamente desarrollado en Java, un lenguaje robusto y ampliamente utilizado para el desarrollo de aplicaciones de escritorio.

IDE: Apache NetBeans Para el desarrollo de este proyecto, se ha utilizado el IDE Apache NetBeans. Este entorno de desarrollo integrado es conocido por su facilidad de uso y su capacidad para manejar proyectos de Java de manera eficiente, proporcionando herramientas poderosas para el desarrollo y depuración del código.

Interfaz Gráfica: JFrame La interfaz gráfica del juego ha sido diseñada utilizando JFrame, una clase de Swing en Java que permite crear ventanas independientes para aplicaciones de escritorio. JFrame es ideal para construir aplicaciones con una interfaz gráfica rica en componentes interactivos, como botones, etiquetas y paneles.

Características del Juego:

Juego Clásico de Totito: El juego sigue las reglas clásicas del Totito, donde dos jugadores alternan turnos para marcar una casilla vacía en una cuadrícula de 3x3. El objetivo es conseguir tres marcas consecutivas en una fila, columna o diagonal.

Interfaz Gráfica Intuitiva: La interfaz está diseñada para ser intuitiva y fácil de usar, permitiendo a los jugadores seleccionar sus movimientos mediante clics en la cuadrícula. Cada casilla del tablero está representada como un botón dentro de la ventana de JFrame.

Gestión de Turnos: El juego implementa un sistema de gestión de turnos para alternar entre los dos jugadores. La clase turno.java probablemente contiene la lógica para determinar de quién es el turno y realizar las verificaciones necesarias después de cada movimiento.

Detección Automática del Ganador: El juego incluye la lógica necesaria para detectar automáticamente cuando un jugador ha ganado, así como para manejar situaciones de empate.

Reinicio del Juego: Una vez que el juego ha terminado, los jugadores tienen la opción de reiniciar la partida y comenzar una nueva, manteniendo la interfaz limpia y preparada para un nuevo enfrentamiento.
